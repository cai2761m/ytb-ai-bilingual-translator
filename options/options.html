YouTube

实时字幕翻译 API

用于 YouTube 字幕抓取、预翻译和视频上的双语字幕叠加层。

Webpage

沉浸式翻译 API

用于普通网页右侧小圆球触发的中英对照翻译。

默认沿用实时字幕 API；选择服务商后可单独配置沉浸式翻译调用的 API。
